# 🎲 Random GIF Finder

A modern React application that generates random GIFs and allows users to search GIFs by custom tags using the Giphy API.

This project demonstrates practical implementation of React Hooks, Custom Hooks, API Integration, Asynchronous Programming, and Component-Based Architecture.

---

## 🚀 Live Demo

🔗 Live Website: https://random-gif1.netlify.app

🔗 GitHub Repository: https://github.com/Kartiksahu03/Random-gif-finder

## 📖 Project Overview

Random GIF Finder is a React-based web application that fetches GIFs dynamically from the Giphy API. Users can generate completely random GIFs or search for GIFs related to specific keywords.

The project was developed to strengthen understanding of:

- React Functional Components
- React Hooks
- Custom Hooks
- API Integration
- Asynchronous JavaScript
- Environment Variables
- State Management
- Deployment Workflow

---

## ✨ Features

### 🎲 Random GIF Generation
Generate completely random GIFs fetched from the Giphy API.

### 🔍 Tag-Based GIF Search
Search GIFs using custom keywords such as:

- Cars
- Nature
- Sports
- Anime
- Technology

and many more.

### ⚡ Dynamic API Integration
Real-time data fetching using Axios and asynchronous JavaScript.

### ♻️ Reusable Custom Hook
Implemented a custom React Hook to manage API requests and loading states efficiently.

### 📱 Responsive Design
Optimized layout for desktop and mobile devices.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Kartiksahu03/Random-gif-finder.git
```

### Navigate to Project

```bash
cd Random-gif-finder
```

### Install Dependencies

```bash
npm install
```

### Create Environment Variable

```env
REACT_APP_GIPHY_API_KEY=YOUR_API_KEY
```

### Run Development Server

```bash
npm start
```
